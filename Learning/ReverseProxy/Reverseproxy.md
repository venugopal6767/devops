what is reverse proxy?
    A reverse proxy is a server that sits between client devices and a backend server, forwarding client requests to the backend server and then returning the backend server's response to the clients.

**Usecases:**
----------
**Load Balancing:** Distributes incoming traffic across multiple servers to ensure no single server is overwhelmed, improving performance and reliability.

**Web Acceleration:** Caches content to reduce the load on backend servers and speed up response times.

**Security:** Acts as an additional layer of defense by hiding the backend servers' IP addresses and can also provide DDoS protection.

**SSL Termination:** Handles SSL encryption and decryption, reducing the processing load on backend servers.

**Centralized Authentication:** Implements authentication once and forwards requests to the appropriate backend server.
